# Saksham Tikekar - Premium Anti-Gravity Portfolio

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Features
- **3D Anti-Gravity Environment**: High-performance 3D scene with an authentic portrait mapping.
- **Gyroscope Tilt Engine**: Mobile-optimized motion tracking for immersive depth.
- **Fluid Typography**: Dynamic scaling using `Space Grotesk` and `clamp()`.
- **Zero-Overlap Layout**: Clean, high-ticket aesthetic with perfect responsive positioning.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- Next.js 16 (Turbopack)
- React Three Fiber / Drei
- Framer Motion / GSAP
- Tailwind CSS
- Space Grotesk Google Font
